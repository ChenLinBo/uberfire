Perspectives

Perspectives provides a powerful mechanisms for task oriented interaction with resources, multi-tasking and information filtering.

Flexible Layout

Users can define, by just dragging and drop, their own layout. This is not only helps usability but also gives users more control of their own work environment.

Powerful Backend

A Metadata Engine, Security Framework and a VFS API (backport of NIO.2) are key components of UberFire Backend.

Open Source

UberFire is developed and mantained by Drools&jBPM team under the Apache License, Version 2.0.

About
More about UberFire...

UberFire is a web based workbench framework inspired by Eclipse Rich Client Platform. This is a very strategic project for Drools & jBPM team, once this is the base technology for our next generation of web tooling.

This project is amazing, and has a bright future... but as an Alpha version we know that it needs some improvements, specially on Look & Feel (no investments have been made on it).
